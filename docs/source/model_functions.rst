*****************************
The model_functions directory
*****************************


The directory *skillmodels.model_functions* contains three types of functions that are relevant to define nonlinear dynamic latent factor models independent of the estimator used to estimate the models: 1) transition functions, 2) anchoring functions and 3) endogeneity correction functions. Users are encouraged to add their own functions to these modules.
